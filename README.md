# Dugsi ERP

School ERP for a secondary school in Somaliland (Form 1–Form 4).

## Stack

- **Backend:** Laravel (PHP)
- **Database:** MySQL
- **Hosting:** Bluehost (planned)
- **SMS:** Telesom SMS gateway (planned)
